# Port Scanner с уведомлениями в ВКонтакте

## 📌 Описание проекта

Данный проект представляет собой программное решение для автоматизированного сканирования сети с целью обнаружения открытых портов и определения сервисов. Программа использует утилиту **Masscan** для быстрого многопоточного сканирования, анализирует баннеры для идентификации сервисов, сохраняет историю обнаруженных портов и отправляет уведомления о новых угрозах через **VK API**.

Проект выполнен в рамках учебного задания и демонстрирует навыки работы с сетевыми технологиями, Python, Docker и API.

---

## 🚀 Функциональность

- ✅ **Многопоточное сканирование IP-адресов** с помощью Masscan
- ✅ **Определение открытых портов** и анализ баннеров
- ✅ **Идентификация сервисов** (SSH, HTTP, SMB, FTP и др.)
- ✅ **Фиксирование новых угроз** — сравнение с предыдущими результатами
- ✅ **Уведомление владельца** через VK API (личные сообщения)
- ✅ **Хранение истории** в JSON-файле
- ✅ **Гибкая настройка** через `config.ini`
- ✅ **Запуск в Docker** для изоляции и переносимости

## ⚙️ Установка и запуск

### 1. Клонирование репозитория
```bash
git clone <repository-url>
cd port_scanner
```

### 2. Настройка конфигурации
Создай файл `config.ini` (см. пример ниже)

```ini
[vk]
token = vk1.a.xxxxxxxx...
user_id = 123456789

[scan]
targets = 192.168.1.1-192.168.1.255
ports = 80,443,22,445,3389
rate = 500
banners = yes

[storage]
history_file = history.json
```

### 3. Запуск в Docker (рекомендуется)
```bash
chmod +x build.sh run_docker.sh
./build.sh
./run_docker.sh
```

### 4. Локальный запуск (без Docker)
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python3 main.py
```

## 📊 Алгоритм работы

1. **Пользователь вводит IP-диапазон и порты** (или использует настройки по умолчанию)
2. **Программа запускает Masscan** для обнаружения открытых портов
3. **Анализируются баннеры** для определения сервисов
4. **Результат сравнивается с историей** (`history.json`)
5. **Если обнаружены новые порты** — отправляется уведомление в ВК
6. **История обновляется** и сохраняется

## 📨 Уведомления в ВКонтакте

Программа использует VK API для отправки сообщений. Для работы необходимо:

1. Создать сообщество ВКонтакте
2. Получить токен доступа в разделе **Управление → Работа с API**
3. Указать `user_id` получателя (свой ID или ID группы)
4. Включить права **Сообщения сообщества**

**Пример уведомления:**
```
🆕 НОВЫЙ ОТКРЫТЫЙ ПОРТ!

📍 192.168.1.65:445
🔧 Сервис: microsoft-ds
```

## 🐳 Запуск в Docker

### Сборка образа
```bash
docker build -t port-scanner .
```

### Запуск контейнера
```bash
docker run --rm \
    --network host \
    --cap-add=NET_RAW \
    --cap-add=NET_ADMIN \
    -v $(pwd)/results:/app/results \
    -v $(pwd)/config.ini:/app/config.ini:ro \
    port-scanner
```
